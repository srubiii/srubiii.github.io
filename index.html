Project Overview:
This project analyzes whether how much cheese a country produces affects the countries netflix prices. The goal of this project is to determine if the cheese diversity of countries has effects on economic factors such as netflix prices.

The two datasets used for this project are the Global Cheese Dataset from Kaggle and the Netflix Subscription Price Dataset from Kaggle. Both are linked below. The Global Cheese Dataset has over 1000 rows of cheese varieties, countries, and textures. The Netflix Price Dataset has cost per month, number of shows/movies, and has three different subscription prices by country. At this time no more datasets are needed.

Extraction, Transform, and Load (ETL):
For this milestone the Global Cheese Dataset was loaded and focused on. The cheese dataset needed tiding when it came to NaN values. Another challenge was how some countries had two or more associated within the column. A dictionary was created for those countries that have multiple values and the broadest range was chosen to match those countries. The Cheese dataset had a lot of NaN values. If the NaN value was in the country column that row was dropped from the dataset because that row will not help answer any of the questions asked above. If any other column had a NaN value it was given the string "unknown".
